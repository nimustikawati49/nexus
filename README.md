# nexus
CYBER NEXUS adalah aplikasi web edukasi inovatif berbasis Single-File Web Module Engine. Menggabungkan kuis interaktif (asesmen formatif/sumatif) dengan elemen gamifikasi RPG bernuansa Neo-Cyberpunk, aplikasi ini memudahkan guru menyusun dan memantau kuis secara real-time lintas gawai tanpa instalasi rumit bagi siswa.
